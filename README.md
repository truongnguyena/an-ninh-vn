# Game An Ninh Mạng Việt Nam

Game chiến thuật hacking đa người chơi.

## 🚀 Một Click Triển Khai Lên Web (Miễn Phí)

Để triển khai game này lên web một cách tự động, bạn chỉ cần thực hiện 2 bước sau:

1. **GitHub**: Đẩy mã nguồn này lên repo của bạn (Sử dụng file `DEPLOY_AUTO.bat` tôi đã tạo).
2. **Render**: Nhấn vào nút dưới đây để tự động nhận cấu hình và chạy game:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/truongnguyena/an-ninh-vn)

---

## 🛠 Cấu hình Offline
Nếu chạy ở máy cá nhân:
1. `npm install`
2. `npm run build`
3. `npm start`
